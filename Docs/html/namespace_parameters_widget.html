Detailed Description

ParametersWidget.

This module defines 2 classes responsible for displaying and handling a list of parameters

General explanation about parameters
  1. There are 4 modules involves in setting the parameters for a class
    1. A class that is the target of the parameters (The class that will use the parameters)
    2. A dialog which holds a ParametersWidget for each class
    3. A ParametersWidget for setting the parameters - This is a general perpose widget that can be used in any dialog
    4. A Parameter module for holding the parameters - This is a general perpose module that can be used for all the parameters
  2. The parameters are initiated in a prms method of the first type
  3. When generating a parameters their has to be a definitions of all the values of the parameter
  4. In Particular 2 methods are defined
    1. A method for generating the widget of the parameter : This is a static method in StdPrmInput.
    2. A method that is the slot of the widget : This is a method in the ParametersWidget. In order to set the method the prms() method gets the widget as a parameter
  5. If you want to program a new behavior of a parameter you should
    1. Create a static method for generating the widget in StdPrmInput
    2. Create a slot method in ParametersWidget
    3. In the prms method create a parameter with these methods
The classes are:
  1. ParametersList - Which is responsible to hold a list of Parameter
  2. ParametersWidget - Which is responsible for presenting the list of Parameter
The process of using this widget is:
  1. In the Qt designer insert the widget to the dialog
  2. The main comboBox holds a list of classes that the widget can present parameters from
    1. Use the method init to set the base class of these classes
    2. The activation of this method should be placed in the init method of the dialog after setupUI
    3. Each class that the widget can present the parameters from should have a prms method that returns the list of Parameter
  3. Use the results method to get the parameters and the main combo box selection
