Detailed Description

This class is an interface to the AlgorithmData which simulate access like regular list.

This class is used when there is a need to access the AlgorithmData and other python lists in the same code

The mechanize used to get and set data

  1. A method from this class is called. It returns a AlgorithmDataRowInterface
  2. It gives the AlgorithmDataRowInterface the AlgorithmData and the row
  3. The AlgorithmDataRowInterface get the index of the column in it's getitem/__setitem__ method
  4. The AlgorithmDataRowInterface composes the index np style and calls the AlgorithmData with this index

Examples

>>> a[1][1]
accessing data in the header row 1 (types)
>>> a[7][1]
accessing the data (starting from row - len(a.algorithm_data.header))
