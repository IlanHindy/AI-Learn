Character Identifier.

The perpose of this module is to demostrate distance metrics

The module allows to create a character set and identify an input character

The module is composed from 3 dialogs:

  1. A main dialog (CharacterIdentifier) which present the character set and allow operations on this set (load, add, remove)
  2. A dialog for entering a character (CharIdentifierInput)
  3. A dialog to show the selection result (CharacterResolveResult)
