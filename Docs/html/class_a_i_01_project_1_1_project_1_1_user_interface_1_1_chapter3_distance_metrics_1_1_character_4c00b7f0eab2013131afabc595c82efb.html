Detailed Description

Main dialog for character identifying.

The dialog allows the following operations:

  1. Load all characters in a directory (characters are files with extention chr)
  2. Select a character
  3. Activate the CharIdentifierInput to add a new character to the character set
  4. Remove the selected character
  5. Resolve a character :
    1. Activate the CharIdentifierInput for getting the character to identify
    2. Select the neerest character from the character set
    3. Present the result in CharactyerResolveResult

The graphical structure of the dialog is :

  1. QGraphicsView which holds a
  2. QGraphicsScene which holds a
  3. CharacterIdentifierItem (inherits from QGraphicsRectItem) which hold many
  4. CharItem (Inherits from QGraphicsRectItem)

Init results If the user press the cancel button when the characters input directory is asked when initiating the member initResults is checked this member should be checked before activating the dialog using exec_, exec, show
