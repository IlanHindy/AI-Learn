Detailed Description

This item represents character item.

The purpose of the class is to draw a character, create a matrix of rectangles and resolve in which rectangles the character passes

The class allow the following operations

  1. Drawing a character using the mouse events:
    1. Start by the mouse press event
    2. Continues by the mouse move event
    3. The character is stored in QGraphicsPathItem
  2. Transform the character to occupy the whole item's space
  3. Set operation : resolving the Occupied matrix which tell on which rectangle the character passes
  4. Reset operation : reverse the character transform so it is possible to continue drawing the character
  5. Save operation : To a QDataStream
  6. Load operation : From a QDataStream

The graphical view of the class is composed from:

  1. This class which inherits from QGraphicsRectItem and holds :
  2. A QGraphicsPathItem : representing the character
  3. A QGraphicsPathItem : representing the occupied rectangles
  4. A QGraphicsPathItem : representing the unoccupied rectangles

def AI Project.Project.UserInterface.Chapter3DistanceMetrics.CharacterIdentifier.CharItem.calcRowCol (   self,
  point 
)

Calculate the network row and column that a point is int calc the row and column indexes of a point.

The following is the algorithm:

  1. Find the distance between the point and the left (or top)
  2. Divide the distance with the width of path to find the relative position
  3. Multipile this relative position with the number of rows/cols
  4. Convert the result to int to find the indexes
  5. If the index is the number of row/col reduce the index (This is for the case the the point is on the boundary and in this case the relative position is 1 which will cause the indexes to be the number of rows/cols - out of the matrix indexes)
Parameters
pointThe point to resolve
Returns
int The network row that the point is in
int The network column that the point is in
