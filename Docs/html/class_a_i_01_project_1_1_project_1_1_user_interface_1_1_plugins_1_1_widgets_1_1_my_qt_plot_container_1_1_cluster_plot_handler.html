Detailed Description

Class ClusterPlotHandler.

This class is a model for MyQtPlotContainer meant to compare the results of the clustering algorithm and the actual clustering

The actual clustering is presented by the colors The result of the clustering algorithm is presented by the shape

In order to avoid recalculation in each phase the following data is saved as members:

  1. The possible results
  2. The result types - means to which cluster in the results an entry belongs
  3. The reduced dimensions - the algorithm parameters reduced dimensions in order to set the place of the shape in the graph
  4. The number of possible results
  5. The total number of combinations (the number of possible results pow 2)
  6. The labels of the options (used for the legend)
