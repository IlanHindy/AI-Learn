Detailed Description

Distance Metrics.

This class implements 3 types of distance metrics between 2 vectors:

  1. Euclidean distance metrics
  2. Manhattan distance metrics
  3. Chebyshave distance metrics

The distance metrics is done between 2 AlgorithmData. at the beginning of the distance metrics methods there is a conversion to AlgorithmData
