Detailed Description

Get a character.

A dialog for creating / getting a character

The followings are the operations in the dialog

  1. Painting a character
  2. Set : analyze the character drawn which is composed from the following stages
    1. Transform the character to the size of the main item
    2. Draw a network composed from squares
    3. Decide which squares are occupied by the character
  3. Save : Saves the character
  4. Load : Loads the character from file
  5. Cancel : exit and retrun False
  6. OK : exit and return True

Graphical structure of the dialog

  1. Vertical box layout
  2. Buttons horizontal layout
  3. GraphicsView which holds a
  4. Graphics scene which hols a
  5. CharItem
