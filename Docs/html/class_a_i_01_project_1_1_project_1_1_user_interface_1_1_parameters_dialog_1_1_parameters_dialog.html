Detailed Description

A dialog for setting parameters for running an algorithm.

Dialog Description

Perpose

There are 3 type of classes involves in the running of an algorithm

  1. The RunningDialog
  2. The train class
  3. The algorithm class

This dialog allows selection of the classes and set their parameters

Structure

  1. 3 ParametersWidget for each type of class
  2. An exit button

Dialog working method

  1. After initializing the ParametersWidget the dialog calls theire init method
  2. When the exit button is pressed each dialog is asked to activate it's save file option
  3. After the dialog was closed the calling dialog can get the parameters using methods of this dialog
